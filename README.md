# CDN Storage Contribution Guide

Welcome! This repository hosts files for our CDN. We accept community contributions!!

## How to Contribute

### Submitting Files via Pull Request

If you'd like to add a file to our CDN storage, please follow these steps:

1. **Fork this repository**
2. **Add your files to the repo**

3. **Create a pull request** with:
   - A clear description of what the file is
   - The intended use case
   - Confirmation that you have the rights to distribute this file
